# lambdasawa/dotfiles

## Usage

```sh
cd ~ && git clone https://git.io/JLdsO && cd ~/.dotfiles && make
```

## Usage (Ubuntu in AWS EC2)

```sh
cd ~ && git clone https://git.io/JLdsO && cd ~/.dotfiles && make start-aws-ec2
```

## Usage (Ubuntu in Vagrant)

```sh
cd ~ && git clone https://git.io/JLdsO && cd ~/.dotfiles && make start-vagrant-ubuntu
```

## Usage (Kali Linux in Vagrant)

```sh
cd ~ && git clone https://git.io/JLdsO && cd ~/.dotfiles && make start-vagrant-kali
```
